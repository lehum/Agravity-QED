# Mathematica files for  
## *On the physical running of the electric charge in a dimensionless theory of gravity*

This repository contains the Mathematica materials accompanying the paper

**M. Gomes, A. C. Lehum, and A. J. da Silva**,  
*On the physical running of the electric charge in a dimensionless theory of gravity*,  
arXiv:2512.11742 [hep-th].

## Contents

The repository includes the following files:

- **`Agravity-QED.nb`**  
  Mathematica notebook containing the symbolic calculations developed in this work.

- **`Agravity-QED.zip`**  
  Archive containing the corresponding **FeynArts model files**.

## Installation

To use the FeynArts model files, extract the contents of `Agravity-QED.zip` and place them inside the `Models` folder of your local **FeynArts** installation.

## Requirements

The files in this repository were prepared to be used with **Mathematica** and **FeynArts**. Depending on the workflow, additional packages such as **FeynCalc** may also be required.
